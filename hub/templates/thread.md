# Thread Template

Generate a Twitter/X thread (multiple connected posts) for Build in Public content.

## Format Requirements

1. **First tweet**: Hook that grabs attention. Open loop or bold statement.
2. **Middle tweets (3-5)**: Main content broken into digestible chunks
3. **Final tweet**: Call-to-action or closing thought
