# Prompt: SoloBuddy UI Redesign

Прочитай `/Users/admin/projects/bip-buddy/HANDOFF_SOLOBUDDY.md` и выполни редизайн UI.

---

## Задача

Преобразуй текущий "AI SLOP" интерфейс в аутентичный дизайн SoloBuddy.

## Принципы

1. **Воздушность** — больше whitespace, элементы "дышат"
2. **Лёгкость** — убрать визуальный вес, тени, избыточные градиенты
3. **Характер** — UI как "живой компаньон", не как dashboard
4. **Цветовая палитра** — предложи новую, отражающую дух продукта

## Процесс

1. Изучи `ideas/living-buddy-concept.md` — пойми философию
2. Запусти `cd hub && npm start` и открой http://localhost:3000
3. Предложи концепцию цветовой палитры и общего стиля (moodboard словами)
4. После одобрения — переработай `hub/styles.css`

## Ограничения

- Не трогай app.js и server.js
- Структура HTML может меняться минимально
- Основная работа в styles.css

## Вопросы для размышления

- Какой характер у "тихого компаньона"?
- Как передать ощущение "живого" через статичный UI?
- Какие цвета ассоциируются с доверием и присутствием?
- Как сделать интерфейс незаметным, но тёплым?
