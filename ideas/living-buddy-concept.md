# 🧬 SoloBuddy — Концепт v0.1

> Гиперперсонализированный AI-компаньон для solo creators.  
> "Около-живое существо, которое дышит когда тебя нет."

---

## ✅ Принятые решения

| Вопрос | Решение |
|--------|---------|
| Где живёт? | **Self-hosted** (локально у юзера) |
| Privacy | **Кастомно** — юзер выбирает уровень |
| LLM | **Claude** (MVP), позже свой fine-tuned когда локалки окрепнут |
| Имя | **SoloBuddy** |

---

## 🎯 Core Philosophy

**Это не ассистент. Это сущность, которая живёт рядом с твоими проектами.**

| Традиционный AI | Living Buddy |
|-----------------|--------------|
| Мёртв пока не спросишь | Дышит в фоне |
| Забывает между сессиями | Помнит и растёт |
| Универсальный | Гиперперсонализированный |
| Реактивный | Проактивный |

---

## 🔌 "Вилки в розетку" — Как проекты подключаются

**Органический онбординг**: проект "просачивается" через работу.

```
[Подключаешь Git] → Buddy начинает впитывать
                          ↓
              Коммиты, паттерны, ритмы
                          ↓
              Постепенно "понимает" проект
```

**Никаких анкет, никаких README.** Buddy сам разбирается.

## 💓 "Дыхание" — Жизнь в фоне

### Источники питания (по приоритету)

| Фаза | Источник | Что Buddy узнаёт |
|------|----------|------------------|
| MVP | Git | Ритм работы, что меняется, настроение коммитов |
| V2 | Соц. сети | Публичный голос, интересы, аудитория |
| V3 | Площадки | Где болтаешь, с кем, о чём |

### Что делает когда тебя нет

- Переваривает новые коммиты
- Замечает паттерны (пушишь ночью? проект заброшен?)
- Готовит инсайты к следующей встрече
- Формулирует вопросы и предложения

## 📬 Инициатива — Как говорит первым

**Модель: iMessage / Telegram**

- Асинхронные сообщения: открываешь — а там уже мысль
- Опционально: push-уведомления для срочного

```
[Ты открываешь Buddy]
         ↓
"Заметил, что последние 3 дня не трогал SPHERE.
 Это пауза или застрял? Могу накидать идей."
```

**Не навязчиво. Умеет молчать.**

## 🏗️ Архитектура (высокий уровень)

```
┌──────────────────┐     ┌──────────────────┐
│   Git Watcher    │────▶│   Brain Core     │
│  (webhooks/poll) │     │  (LLM + Memory)  │
└──────────────────┘     └────────┬─────────┘
                                  │
┌──────────────────┐              │
│  Social Listener │──────────────┤
│  (Twitter, etc)  │              │
└──────────────────┘              ▼
                         ┌──────────────────┐
                         │   Interface      │
                         │  (Web/Desktop)   │
                         └──────────────────┘
```

### Ключевые компоненты

| Компонент | Роль |
|-----------|------|
| **Git Watcher** | Слушает репозитории, парсит коммиты |
| **Brain Core** | LLM + долгосрочная память + личность |
| **Message Queue** | Асинхронные сообщения от Buddy к тебе |
| **Interface** | Чат + статус проектов |

## 📊 V2 как продукт (Стартап)

### Аудитория

**Solo creators:**
- Инди-разработчики
- Дизайнеры-одиночки  
- Контент-криейторы
- Художники/музыканты с техническими проектами

### Монетизация (идеи)

| Модель | Описание |
|--------|----------|
| Subscription | $X/month за полного Buddy |
| Freemium | Бесплатный с ограничениями (1 проект, базовая память) |
| Pay-per-project | Каждый "слот" — отдельная оплата |

### Масштабирование

- Каждый Buddy — изолированный instance
- Brain Core может быть serverless (OpenAI API / Claude API)
- Память — vector DB per user (Pinecone, Weaviate)
