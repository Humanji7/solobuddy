## Emergency Fixes

### Tweet underperforming?
1. Check timing (repost Tu-Th 8-11AM)
2. Add video version (2-4x boost)
3. Rewrite hook (use curiosity gap)
4. Engage actively in first 2h
5. Convert to thread (3-5 tweets)

### Getting Community Notes?
1. Delete immediately
2. Fact-check thoroughly
3. Repost with sources
4. Wait 48h+ before related topic

### Shadow banned?
1. Stop all automation
2. Remove engagement bait
3. Post high-quality only (2-3/day)
4. Engage genuinely (no spam)
5. Wait 7-14 days
