# Commands

## Hub Server

```bash
# Start development server
cd hub && npm start

# Install dependencies (first time)
cd hub && npm install
```

Server runs at: http://localhost:3000

## Git

```bash
# Standard commit
git add -A && git commit -m "message"

# Push to GitHub
git push origin main
```

## Environment

Required: `hub/.env` with GitHub OAuth credentials.
Template: `hub/.env.example`
